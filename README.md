# 📝 Personal Weblog - nandcep.github.io
Selamat datang di repositori catatan pribadi saya. Situs ini adalah wadah bagi saya untuk mendokumentasikan pemikiran, pengalaman, dan refleksi harian dalam bentuk tulisan digital.

🌐 Kunjungi Situs: [nandcep.github.io](https://nandcep.github.io)

# 💡 Filosofi Konten
Semua konten yang dipublikasikan di sini bersifat independen dan subjektif.

1. Pengalaman Murni: Tulisan berdasarkan apa yang saya alami dan rasakan secara langsung.
2. Bukan Teori Buku: Meskipun mungkin ada kemiripan dengan teori yang ada, perspektif yang diambil murni dari sudut pandang pribadi.
3. Tanpa Afiliasi: Seluruh isi tulisan tidak mencerminkan kebijakan, opini, atau pengaruh dari perusahaan tempat saya bekerja saat ini maupun mantan perusahaan di masa lalu.
